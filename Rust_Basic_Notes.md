


Debug print macros

{:?} will debug print the contents

------Regular-----
Student { name: "Lidiya", gpa: 3.5 }
Student { name: "Kyle", gpa: 3.9 }
Student { name: "Anatoly", gpa: 4.0 }


{:#?} will debug print the contents in human readable form


------Human Readable-------
Student {
    name: "Lidiya",
    gpa: 3.5,
}
Student {
    name: "Kyle",
    gpa: 3.9,
}
Student {
    name: "Anatoly",
    gpa: 4.0,
}
